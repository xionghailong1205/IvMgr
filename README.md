# IvMgr

IvMgr 是一个用于管理版本、提交和日志的命令行（CLI）工具。

## 命令

- `ivmgr init` - 初始化项目
- `ivmgr commit` - 提交更改,打上 Tag.
- `ivmgr push` - 推送代码到远程仓库
- `ivmgr check` - 检查 DevLog 文件格式
- `ivmgr reset` - 回退到上一次提交前的状态 去除 tag git reset 到上一个提交
- `ivmgr version` - 查看工具版本

## 脚本

- `npm run build` - 编译构建项目
- `npm run dev` - 运行开发模式（监视文件更改）
